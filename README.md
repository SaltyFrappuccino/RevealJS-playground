# Вариант №2. Численные методы

Презентация выполнена на Reveal.js с локальными зависимостями, Markdown-слайдами, отдельной CSS-темой и интерактивной JavaScript-программой для решения СЛАУ.

## Запуск

```powershell
bun install
bun run start
```

Открыть в браузере:

```text
http://localhost:4173
```

Проверка локальных подключений:

```powershell
bun run check:local
```

## Задание

Презентация должна быть выполнена с помощью Markdown, где это возможно.

Презентация должна быть выполнена с помощью фреймворка Reveal.js.

Тема оформления обязательно должна быть вынесена в отдельный CSS-файл.

Минимальное количество слайдов: 20.

Перед каждым разделом необходимо использовать слайд-разделитель.

Оформление презентации должно быть читаемым.

Обязательными являются титульный слайд и заключительный слайд.

Обязательно наличие меню для быстрого перехода к слайду с использованием плагина reveal.js-menu. Меню должно быть рабочим. Все слайды должны иметь наименование в меню.

Обязательно наличие навигации: стрелки и номер слайда.

Обязательно наличие README в корне презентации с заданием.

В случае сомнения самостоятельности выполнения работы требуется обоснование спорных мест.

На слайде не должно быть скроллов.

Минимальное количество изображений в презентации: 3.

После создания репозитория запрещается его удалять и перезаливать презентацию в новый репозиторий. Если есть проблемы, необходимо обращаться за помощью.

Необходимо использовать локальную версию фреймворка Reveal.js.

Максимальная дата сдачи презентации: 1 июня 2026 года.

Определение численных методов.
